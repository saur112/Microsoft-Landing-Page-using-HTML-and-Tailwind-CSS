# Microsoft-Landing-Page-using-HTML-and-Tailwind-CSS
Created a Clone of Microsoft Landing Page using Html and Tailwind CSS
To run the file follow below instructions:
1. Firstly download the file and open in code editor
2. Now open the terminal and make sure that your in root direrctory where your index.html file is
3. finally run "npm install" to install node module
4. And to run file in browser run "npm run start"

Dependencies:
1. Node must be installed
2. Tailwind CSS must be installed (npm install -D tailwindcss postcss autoprefixer in order to use tailwind from .css file) 
3.Install tailwindcss and its peer dependencies via npm, and create your tailwind.config.js file.
